# ByteWise Nutritionist - Live App

🍎 **Professional Nutrition Tracking Progressive Web App**

This is the live application for ByteWise Nutritionist. For source code and documentation, visit the main repository.

## Features
- USDA Database Integration
- Smart Meal Logging
- Weekly Progress Tracking
- PWA Functionality (Installable)
- Offline Support
- Professional UI/UX

## Install as App
1. Visit this page on your mobile device
2. Tap "Add to Home Screen" or "Install"
3. Use ByteWise like a native app

Built with React, TypeScript, and Supabase.
