Food Logging Process

1
Search for Food
Type the name of any food item. The app searches the USDA database for accurate nutrition data.
2
Select Food Item
Choose from search results. Enhanced candy items show detailed nutrition information.
3
Choose Serving Size
Select appropriate serving size or enter custom amount.
4
Auto Meal Detection
App automatically detects meal type based on time:
  • Breakfast: 6:00 AM - 11:00 AM
  • Lunch: 11:00 AM - 4:00 PM
  • Dinner: 4:00 PM - 10:00 PM
  • Snack: 10:00 PM - 6:00 AM
Accurate Calculations: The app uses the standard 4-4-9 method (4 calories per gram of protein/carbs, 9 calories per gram of fat) for precise calorie calculations.

üîß Troubleshooting & Help

Food Search Issues +
  • Check your internet connection
  • Try different search terms or keywords
  • Clear browser cache and refresh the page
  • Use specific food names rather than brand names
Try searching for "chicken breast" instead of "Tyson chicken breast" for better results.
Login Problems +
  • Verify your email address is correct
  • Check that password meets requirements
  • Look for verification emails in spam/junk folders
  • Use the "Reset Password" link if needed
Remember: Passwords must contain uppercase, lowercase, numbers, and special characters.
Data Sync Issues +
  • Refresh the page to reload data
  • Check internet connection stability
  • Sign out and sign back in
  • Contact support if issues persist
Nutrition Data Questions +
  • Verify you selected the correct food item
  • Check serving size matches consumption
  • Use enhanced candy data for accurate candy nutrition
  • Try alternative food descriptions if needed
Data Sources: All nutrition data comes from official USDA databases with enhanced candy information from FoodStruct.com for maximum accuracy.
